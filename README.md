# CopyLab Python SDK

A Python SDK for integrating CopyLab features into your applications.

## Installation

### Via pip with Git

You can install the package directly from GitHub:

```bash
pip install git+https://github.com/nac5504/CopyLab-Python.git
```

### Manual Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/nac5504/CopyLab-Python.git
    ```
2.  Navigate to the directory:
    ```bash
    cd CopyLab-Python
    ```
3.  Install dependencies:
    ```bash
    pip install .
    ```
